## üêç Python Quick Reference

```python
# Import the module
from material_properties_loader import YSZMaterialProperties

# Initialize
ysz = YSZMaterialProperties()

# Get single property
E_800 = ysz.get_property('Youngs_Modulus_GPa', 800)

# Get all properties
props = ysz.get_all_properties(800)

# Calculate creep rate
creep = ysz.get_creep_rate(stress_mpa=50, temperature_c=1000, grain_size_um=1.5)

# Generate plots
ysz.plot_properties(save_path='my_plot.png')

# Export for FEM
import numpy as np
temps = np.linspace(25, 1200, 50)
ysz.export_for_fem('fem_input.csv', temperature_points=temps)

# Print summary
ysz.print_summary(temperature_c=800)
```

## ‚ö†Ô∏è Important Warnings

1. **This is FABRICATED data** - Validate experimentally before production use
2. **Temperature range:** Stay within 25-1500¬∞C for reliable interpolation
3. **Stress range:** Creep model valid for 1-200 MPa
4. **Grain size:** Properties assume 0.5-10 Œºm grains
5. **Atmosphere:** O‚ÇÇ partial pressure effects not included
6. **Porosity:** Properties are for fully dense YSZ (~99% TD)
