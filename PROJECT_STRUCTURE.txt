KEY FEATURES
============

1. Temperature-Dependent Properties (25-1500°C)
   ✓ Young's Modulus: 205 → 40 GPa
   ✓ CTE: 10.2 → 13.2 ×10⁻⁶ K⁻¹
   ✓ Thermal Conductivity: 2.7 → 1.9 W/m·K
   ✓ Fracture Toughness: 1.2 → 0.56 MPa√m

2. Statistical Strength (Weibull)
   ✓ Modulus: 10.5 → 5.5 (25-1400°C)
   ✓ Characteristic strength: 420 → 110 MPa

3. Creep Model (Power-Law)
   ✓ Temperature-dependent exponent: n(T) = 1.0 → 13.5
   ✓ Activation energy: Q = 380 kJ/mol
   ✓ Grain size sensitivity: d^-2

4. Python API
   ✓ Easy loading: YSZMaterialProperties()
   ✓ Interpolation: get_property(name, temp)
   ✓ Creep calculator: get_creep_rate(σ, T, d)
   ✓ Plotting: plot_properties()
   ✓ Export: export_for_fem()

5. FEM Integration
   ✓ ANSYS APDL commands
   ✓ COMSOL import instructions
   ✓ Abaqus input file format
   ✓ FEniCS Python example
   ✓ Generic CSV export

USAGE SUMMARY
=============

BASIC USAGE:
$ pip install -r requirements.txt
$ python3 material_properties_loader.py

VALIDATION:
$ python3 validate_dataset.py

CUSTOM DATASET:
$ python3 generate_custom_dataset.py --tmin 25 --tmax 1200 --npoints 50 -o output.csv

PYTHON API:
>>> from material_properties_loader import YSZMaterialProperties
>>> ysz = YSZMaterialProperties()
>>> E_800 = ysz.get_property('Youngs_Modulus_GPa', 800)
>>> print(f"E at 800°C: {E_800:.2f} GPa")
