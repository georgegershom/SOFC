======================================================================
SOFC EXPERIMENTAL DATA SUMMARY
======================================================================

This synthetic dataset contains comprehensive experimental measurements
for Solid Oxide Fuel Cell (SOFC) characterization.

1. DIGITAL IMAGE CORRELATION (DIC) DATA
----------------------------------------------------------------------
   - Sintering strain data (1200-1500°C)
   - Thermal cycling data (ΔT = 400°C, 10 cycles)
   - Startup/shutdown cycle data (5 cycles)
   - Speckle pattern images with timestamps
   - Lagrangian strain tensor outputs (Vic-3D format)
   - Strain hotspot catalog (>1.0% strain)

2. SYNCHROTRON X-RAY DIFFRACTION (XRD) DATA
----------------------------------------------------------------------
   - Residual stress profiles across cross-sections
   - Lattice strain measurements (25-1500°C)
   - Sin²ψ method peak shift data
   - Microcrack initiation threshold data (ε_cr > 0.02)

3. POST-MORTEM ANALYSIS DATA
----------------------------------------------------------------------
   - SEM crack density analysis (cracks/mm²)
   - EDS line scans (Ni, Zr, Y, O, La, Sr, Mn)
   - Nano-indentation maps:
     * Young's modulus (YSZ: 184.7 GPa, Ni-YSZ: 109.8 GPa)
     * Hardness and creep compliance
     * Load-displacement curves
