KEY FIELDS IN DATA FILES
=========================

STRESS MEASUREMENTS (experimental_data/residual_stress/*.csv):
  - Position: x_position_mm, y_position_mm, z_position_mm
  - Stress: sigma_xx_MPa, sigma_yy_MPa, sigma_zz_MPa
  - Shear: sigma_xy_MPa, sigma_xz_MPa, sigma_yz_MPa
  - Derived: von_mises_MPa, hydrostatic_MPa
  - Uncertainty: measurement_error_MPa or stress_uncertainty_MPa

CRACK DATA (experimental_data/crack_analysis/*.csv):
  - Geometry: crack_length_um, crack_width_nm, crack_orientation_deg
  - Critical: critical_load_N, critical_temperature_K
  - Fracture: stress_intensity_factor_MPa_sqrt_m, J_integral_J_m2
  - Type: grain_boundary_crack, pore_associated

FEM OUTPUT (simulation_output/full_field/*.csv):
  - Coordinates: x_coord_mm, y_coord_mm, z_coord_mm
  - Thermal: temperature_K
  - Displacement: displacement_x/y/z_um
  - Stress: stress_xx/yy/zz/xy/xz/yz_MPa
  - Strain: strain_xx/yy/zz/xy/xz/yz
  - Derived: von_mises_MPa, hydrostatic_MPa

COLLOCATION POINTS (simulation_output/collocation_points/*.csv):
  - All FEM fields PLUS:
  - location_type: free_surface / interior / grain_boundary
  - distance_to_pore_um
  - distance_to_grain_boundary_um
  - selection_criterion


INTENDED USE CASES
==================

1. FEM MODEL VALIDATION
   Compare simulation_output/*.csv with experimental_data/residual_stress/*.csv
   → Calculate residuals, identify regions needing refinement

2. SURROGATE MODELING
   Train on: simulation_output/collocation_points/*.csv (30 points)
   Validate on: simulation_output/full_field/*.csv (full solution)
   → Neural Networks, Gaussian Processes, POD-RBF

3. CRACK PREDICTION
   Features: Stress/strain from simulation_output
   Target: Crack locations from experimental_data/crack_analysis
   → Train ML model to predict failure

4. MULTI-SCALE MODELING
   Macro → Meso → Micro hierarchy
   Use RVEs to homogenize meso-scale properties for macro model

5. RESIDUAL ANALYSIS
   Identify where |FEM - Experiment| is large
   → Guide adaptive mesh refinement, physics improvements
